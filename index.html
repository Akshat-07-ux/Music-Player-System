About This Website

This is a website which is a clone of a music playing system. This website is designed through the concepts of HTML, CSS & JAVASCRIPT.
This website displays 10 songs of the THE GOLDEN ERA of Indian Music. On clicking the play button the first song will be played. On clicking the next button which is beside the play/pause button, the next song in the queue will be played. The user can anytime play or pause the song being played by clicking on the play/pause button at the bottom as well as in the white bar also where the song name is displayed. Overall the designing of this website is a result of application of various concepts of Html and css along with the implementation of rules of javascript. The basic structure of the webpage was constructed using the logics of Html. The designing part is completed by using the features of css. And the dynamuc part of the website which includes the plying of songs on clicking the necessary icons was completed using javascript.
